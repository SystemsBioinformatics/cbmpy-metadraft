Installation
============

Install CBMPy (http://www.cbmpy.net)

Copy MetaDraft to your home directory and to run it:

python metadraft.py

If you have any problems with it just apparently hanging for a while, 
look in the help-->syslog menu for error messages. Or in the log subdirectory.
